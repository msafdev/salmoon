## Intro

MDX lets you blend Markdown with JSX, which means you can embed dynamic content—like music, videos, or GitHub repos—directly into your posts using custom components.

This post demonstrates how to embed rich media in MDX using syntax like `::component{}`. Each section below shows a working example along with the type of content you can include.
